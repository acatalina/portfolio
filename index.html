Profile

Hello there! My name is Alvaro and I am a junior full-stack developer. I started coding on February 2017 after 10 years of radiography when my world and life changed for good.

I have been working with html, css (vanilla css, scss, bulma, bootstrap and materialize), javascript (vanilla JS, react, postgresSQL, mongoose, express, EJS...), SQL, mongoDB and learning a little bit of java on my free time.

You will see in my portfolio some projects done with react, express, mongoose... Also, I have included in the contact section links to my github profile and codewars where you can see some of my code.
